## Overview

A desktop application that automates the computer engineering enrollment process, featuring an intuitive graphical user interface, automated report generation, and full lifecycle management for student enrollments.

## Key Features

- **Intuitive GUI**: Clean JavaFX interface with SceneBuilder-designed forms for easy data entry
- **Automated reports**: Generate enrollment summaries, class rosters, and statistical reports
- **Full lifecycle management**: Track students from application through enrollment, course assignment, and graduation
- **Data validation**: Built-in validation rules to ensure data integrity across all enrollment records

## Technologies Used

| Technology | Purpose |
|-----------|---------|
| Java | Core application logic |
| JavaFX | Desktop GUI framework |
| MySQL | Relational database |
| SceneBuilder | Visual layout designer |
