## Overview

A desktop weighing application developed during a technical internship at **Les Conserveries Marocaines Doha** (Ait Melloul), designed to digitize and streamline the recording of production weighing data, replacing manual processes with a reliable and user-friendly tool.

## Key Features

- **Weighing data recording**: Real-time capture and storage of weighing measurements tied to production batches
- **User training & support**: Hands-on guidance for production staff on using the application in daily operations
- **Report generation**: Automated production reports built with CrystalReport for traceability and quality control
- **Machine maintenance awareness**: Active involvement in ensuring weighing machine uptime and production continuity

## Technologies Used

| Technology | Purpose |
|-----------|---------|
| C# | Application logic and backend processing |
| WinForms | Desktop GUI development |
| Microsoft SQL Server | Data storage and management |
